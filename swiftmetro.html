Frequently Asked Questions

Everything you need to know

How do I use my license key?

After purchasing, you'll receive your license key. Open your iOS project in Xcode and find your main app file (the one with @main). It's usually called YourAppNameApp.swift or AppDelegate.swift.

Step 1: Add the import at the top:

import SwifMetro

Step 2: Add this code in the init() section:

@main
struct YourApp: App {
  init() {
    SwifMetroClient.shared.start(
      serverIP: "192.168.0.100",
      licenseKey: "SWIF-XXXX-XXXX-XXXX"
    )
  }
}

Replace 192.168.0.100 with your Mac's IP (shown when you run the server) and paste your license key.

Yes! One license key works across all your iOS projects. Use it on as many apps as you want.
